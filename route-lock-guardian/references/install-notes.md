# Install Notes

Copy the entire `route-lock-guardian` folder to a supported skill location.

Examples:

```text
.agents/skills/route-lock-guardian/
.claude/skills/route-lock-guardian/
.trellis/skills/route-lock-guardian/
```

The supporting folders are part of the skill:

```text
templates/   contract templates copied into projects
references/  review gates, smells, prompt patterns
scripts/     optional helper scripts
examples/    generic example artifacts
```

Do not install only `SKILL.md`; the folder assets are intentionally used by the skill.
